# Ferns N Petals Sales Analysis Dashboard (Excel)

## Project Overview

This project presents an interactive Sales Analysis Dashboard developed using Microsoft Excel, focused on analyzing the business performance of Ferns N Petals.

The dashboard provides deep insights into sales trends, customer behavior, product performance, and delivery efficiency, enabling better data-driven decision making.

## Objectives
The main objectives of this project are:

• Analyze overall sales performance

• Track customer purchasing behavior

• Identify top-performing products and categories

• Understand seasonal and occasion-based trends

• Evaluate delivery efficiency

• Provide actionable business insights

## Dashboard Features
### Key Performance Indicators (KPIs)
• Total Orders: 1000

• Total Revenue: 3.52M

• Average Customer Spending

• Average Delivery Time

### Sales Analysis
• Monthly revenue trends to identify seasonality

• Hourly revenue distribution to analyze peak order times

### Occasion-Based Analysis
Revenue distribution across major occasions:

• Anniversary

• Birthday

• Diwali

• Holi

• Raksha Bandhan

• Valentine’s Day

### Category Analysis
Performance of different product categories:

• Cakes

• Colors

• Mugs

• Plants

• Soft Toys

• Sweets

### Product Insights
• Top 5 products by revenue

• Identification of best-selling items

### Geographical Analysis
• Top 10 cities contributing to revenue

• Regional sales performance insights

### Delivery Performance
• Analysis of average delivery time

• Time gap between order date and delivery date

### Interactivity
Slicers implemented for:

• Order Date

• Delivery Date

• Occasion
